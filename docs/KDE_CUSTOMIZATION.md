# KDE Plasma Kubuntu-like Customization

This guide helps you make KDE Plasma look and feel
similar to Kubuntu after debloating Linux Mint.

## Recommended Settings
- Global Theme: Breeze Dark
- Icons: Breeze
- Window Decorations: Breeze
- Panel: Bottom panel (Kubuntu style)

## Optional Packages
```bash
sudo apt install kdeplasma-addons
